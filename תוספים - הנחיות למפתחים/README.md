# קבצים למפתחי תוספים לאוצריא

תיקייה זו מרכזת את כל הקבצים הרלוונטים ליצירת תוספים (add-ons) לתוכנת אוצריא, שנאספו מהאשכול בפורום:
https://otzaria.org/forum/topic/1453/המלצה-ריכוז-הוראות-למפתחי-תוספים

## הקבצים בתיקייה

- **הנחיות לAI עבור פיתוח תוספים לאוצריא.md** — מדריך גרסה 1.0, הועלה ע"י המשתמש "מיכאלוש" בפורום.
- **הנחיות לAI עבור פיתוח תוספים לאוצריא v2.md** — מדריך גרסה 2.0 (ניסוח מעודכן, תגי `[חובה]` בתוך המשפט).
- **הנחיות לAI עבור פיתוח תוספים לאוצריא V2 (variant).md** — גרסת ביניים נוספת של v2.0 (תגי `[חובה]` בתחילת שורה בטבלאות). ההבדלים בין שני קבצי ה-v2 הם קוסמטיים בעיקרם.
- **plugin-sdk (רשמי מהריפו)/** — התיעוד הרשמי של ה-SDK, שנשלף במישרין מריפו הקוד של אוצריא ([Otzaria/otzaria](https://github.com/Otzaria/otzaria), branch `dev`, נתיב `docs/plugin-sdk/`):
  - `README.md`, `API_REFERENCE.md`, `COOKBOOK.md`, `DESIGN_GUIDE.md`, `otzaria_plugin.d.ts` (הגדרות ה-API ב-TypeScript, מוזרקות ע"י ה-host כאובייקט הגלובלי `Otzaria`).
- **otzaria-plugin-validator (כלי אימות)/** — כלי CLI רשמי לאימות/אריזה של תוספים לפני פרסום ([Otzaria/otzaria-plugin-validator](https://github.com/Otzaria/otzaria-plugin-validator)), כולל תבניות דוגמה תקינות/לא תקינות ב-`test/fixtures/`.

## נקודות מפתח מהדיון בפורום (לא קבצים, אך חשוב לדעת)

מתוך חמשת האשכולות שאליהם הפנה יאיר דניאל באשכול הריכוז:

1. **[איך יוצרים תוסף?](https://otzaria.org/forum/topic/1117)** — תהליך מומלץ: לגבש רעיון מתוך היכרות עם ה-API → להעלות ל-AI (Claude/Gemini) את שלושת קבצי ה-MD + פרומפט ברור, במצב "Pro" → לפתח ב-fork עם דיבוג (בלי לארוז) → לבדוק → לארוז ולפרסם. מומלץ להשתמש ב-IDE ייעודי (למשל Kiro) לדיוק רב יותר.
2. **[אין להציג שם המפתח בתוך חלון התוסף](https://otzaria.org/forum/topic/1451)** — יש להשתמש בכפתור "אודות" במקום להציג את שם המפתח בכותרת/בתוך חלון התוסף (חוסך מקום, מקצועי יותר).
3. **[חובת תמיכה בגלילה](https://otzaria.org/forum/topic/1362)** — WebView אינו גולל אוטומטית; יש להגדיר גלילה במפורש (ראו סעיף SCROLLING ב-`otzaria_plugin.d.ts`). כן נדרש להוסיף צילומי מסך לתוסף.
4. **[שימו לב למפתחים](https://otzaria.org/forum/topic/1344)** — שדות חובה ב-manifest.json; הסרת הרשאות לא קיימות; באג טיימזון עם `toISOString()` (יש להשתמש בפונקציית `localDateStr()` מקומית); חובת sanitize (פונקציית `esc()`) למניעת XSS על כל תוכן שלא נכתב ישירות בקוד.
5. **[חנות תוספים לאוצריא](https://otzaria.org/forum/topic/1214)** — קיימת חנות תוספים רשמית: `https://otzaria.org/plugins` (העלאה: `https://otzaria.org/plugins/upload`), עם תכנון להתקנה ישירה מהחנות ולמצב אופליין.
